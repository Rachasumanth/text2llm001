# `text2llm pairing`

Approve or inspect DM pairing requests (for channels that support pairing).

Related:

- Pairing flow: [Pairing](/channels/pairing)

## Commands

```bash
text2llm pairing list whatsapp
text2llm pairing approve whatsapp <code> --notify
```
