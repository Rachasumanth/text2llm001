# `text2llm tui`

Open the terminal UI connected to the Gateway.

Related:

- TUI guide: [TUI](/web/tui)

## Examples

```bash
text2llm tui
text2llm tui --url ws://127.0.0.1:18789 --token <token>
text2llm tui --session main --deliver
```
